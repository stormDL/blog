## 位(bit)

- 位是二进制中的一位，是计算机存储信息的最小单位.
- 一个二进制位可以表示两种状态

## 字节(byte)

- 计算机信息计量单位，不分数据类型
- 1byte == 8 bit

## 为什么计算机的存储单位的进率是1024而不是1000

因为目前的计算机都是基于二进制，他们计算的时候只有2的整数幂才能非常方便计算机计算．

## 几个单位的换算

1byte == 8bit

1kb == 1024 byte

1mb == 1024kb

1gb == 1024mb

1tb == 1024gb

1pb == 1024tb

1eb == 1024pb

1zb == 1024eb

1yb == 1024zb

1bb == 1024yb
